### About

* Study project on PHP "Brain Games"

## Prerequisites

* Linux, Macos, WSL
* PHP >=8.1.2
* Make
* Git

## Setup

```bash
git clone https://github.com/Streinge/php-project-45.git
cd php-package
make install
```
